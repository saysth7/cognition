### 社会认知基础
- 个体的价值追求、个人能力、社会环境
  + 能力 包括 性格等，def:追求价值所需素质的强弱。

- 价值追求
  + 自身的生活
  + 对环境的好奇心的满足
  + 尽力影响和改造环境

- 个体能力
  + 性格是什么？
  + 硬实力怎么定义？
  + 软实力怎么定义？
    - 毅力、面对失败。。。

- 个体状态 怎么描述？
  + 心情应该有多少种类？常见哪些种类？
    - 面对未知
    - 面对结果
    - 面对评价
    - 面对任务/目标（各种类型的任务/目标）

- 价值目标可以怎么分类？
  + 按：难度、繁琐；按时间轴，基元任务 耗时短、耗时长，组合任务（耗时长）：平均型、毛刺型、等等更多可能的组合、无限期型（能到啥程度就算啥程度）。
  + 评价分类：01型、程度型

- 个体差异与相互看待

- 环境
  + 真实环境
  + 个体心中的环境
  + 个体嘴出的环境

  + 个体描述环境时，分几方面？
  + 个体认识环境有几类方法？

- 结果
  + 成功失败的原因，可归为哪几方面？
    - 未知的上天
    - 环境
    - 个人
      + 硬实力
      + 心态性格
