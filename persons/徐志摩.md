### 部分时间节点
+ 1897 出生
+ 1915 结婚-张幼仪
+ 1918 赴美、学士
+ 1921 赴英，认识林徽因
+ 1922 离婚-张幼仪
+ 1922 回国
+ 1924 新月社创立，新月诗派始成，有胡适、林徽因、梁启超等人
+ 1924 泰戈尔访华，和林徽因同任翻译
+ 1924 陪同泰戈尔访日，作《沙扬娜拉》
+ 1924 创作《雪花的快乐》
+ 1925 林徽因父亲林长民讨伐张作霖身亡、受梁家周济
+ 1926 结婚-陆小曼
+ 1926 《偶然》：有说是写的林徽因
+ 1928 林徽因、梁思成结婚
+ 1928 创作《再别康桥》《我不知道风是在哪一个方向吹》
+ 1931 失事，《新月》诗刊停刊
+ 1933 《新月》解散
+ 1940+ 林徽因西南避战时重度肺病

### 主要事件
+ 徐志摩家世
  - 还算比较富裕啊，镇上首富
+ 与张幼仪婚姻
  - 育有两子
  - 徐志摩只为应付封建要求
+ 与陆小曼婚姻
  - 陆小曼丈夫系徐同门师弟，受托照顾陆小曼
  - 双方家庭都反对
  - 梁启超 致婚词 严厉批评
  - 不能生育
  - 陆小曼婚后奢侈，徐志摩努力挣钱
  - 徐志摩逝后生病、抽鸦片、与有妇之夫同居，但事业正常
+ 林徽因的评价
  - 徐志摩爱的是他想象中的林徽因
  - 《纪念志摩去世四周年》
