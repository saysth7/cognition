+ 这里，既然名利并提，那利就不包括名誉，只包括钱权色这些。
+ 天朝的人老是讲，一个人，如果不追求利，那必定是追求名。
+ 我以为不然：名利之外，还可以有自己的理想，比如一种建立一种理论、推进一种理念，等等。
+ 区别是否追求名誉的方法，我觉得主要是：看那人是任人凭说，还是介意甚至控制他人的评论。
+ 不为名利，只为理想的人，当然很多，比如：
  - 孔子：
    + 孔子的追求在于实现他的等级分明、安分守己、井然有序的社会理念。虽然我不喜欢他这理念，也不为他这种大同理想能实现。
    + 对于别人对自己的理解和评价，
      - 怕所有人都不理解自己
        + “莫我知也夫！”
        + “知我者其天乎！”
      - 但是不怕有人不理解自己
        + “人不知而不愠，不亦君子乎？”
        + “不患人之不己知，患不知人也。”
      - 不怕有人说自己坏话
        + 子路反对他见卫灵公夫人 一事。
      - 但怕别人称赞自己过了头
        + “君子疾没世而名不称焉。”
